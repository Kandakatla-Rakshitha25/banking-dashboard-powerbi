## images/
Place dashboard screenshots here for the README preview.

Recommended screenshots:
- `dashboard-preview.png` — Main overview page (used in README)
- `loan-analytics.png` — Loan analytics page
- `customer-insights.png` — Customer segmentation page
- `branch-performance.png` — Branch map and comparison

**How to take a screenshot in Power BI Desktop:**
1. Go to the page you want to capture
2. File → Export → Export to PDF (for full quality)
3. Or use Windows Snipping Tool / Snip & Sketch
